## About

Elysia Icons is a skeuomorphic style icon theme forked from the Mint-X Icons.

It continues the evolution of classic Linux desktop icon themes from:

- Mint-X Icons
- Faenza Icons


Elysia Icons inherits from this lineage and aims to preserve its spirit while (hopefully) improving consistency and coverage at a faster rate.

It is not meant to replace upstream work, but to extend and refine it for personal and community use.
